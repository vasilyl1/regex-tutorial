# Regex Tutorial : matching an Email

Regex is a short for regular expression. When someone sees the Regex pattern at first, it may seem to be as a cryptic expressions. However, things change as soon as people start to understand how versatile and helpful the Regex can be when it is being used to define a specific search pattern.
So, formally Regex is defined as the expression to be used to find pattenrs of characters within a string, find and replace a character or sequence of characters within a string, or to validate an input if it is a string.

## Summary

This tutorial is aimed to describe one of the most commonly required expressions - matching an Email, which the author of this manual has been able to use a couple of times in the code while he was writing that manual.
The Regex for matching Email is following:

/^#?([a-f0-9]{6}|[a-f0-9]{3})$/
